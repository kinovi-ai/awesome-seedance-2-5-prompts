## 🤔 什麼是 Seedance 2.5？

Seedance 2.5 是字節跳動新一代 Seedance 影片模型，面向更長、更丰富、更高清、更可控的 AI 影片製作。它適合電影感提示詞、產品影片、多參考素材場景和局部可控影片編輯。你可以在 Seedance 2.5 AI 影片生成器中嘗試這些工作流。

**核心能力:**
- 🎬 **30 秒原生影片 — 為廣告、產品演示、教育短片和敘事片段生成更長的單條影片。**
- 🧩 **丰富的多模態參考 — 用人物圖、產品素材、風格板、運動影片、音頻方向、品牌資產和 3D 參考來約束生成。**
- ✂️ **可控影片編輯 — 在保持整體鏡頭穩定的同時調整背景、替換產品、更換模特或精修局部細節。**
- 📐 **面向生產的質量 — 強調更平滑的運動、更強的角色一致性、更好的提示詞遵循能力和 4K 生產工作流程。**
- 🔁 **靈活工作流 — 支持文生影片、圖生影片、影片到影片、多參考生成、首尾幀關鍵幀和定向編輯。**

```

緊貼動感BGM律動剪輯（卡點轉場），頂級電影感調色（Cinematic Color Grading）。逼真的玻璃折射與透射材質，復雜光線追蹤（Ray tracing），全局照明。主體極致清晰，背景帶有強烈的動態模糊，視覺衝擊力極強。
```
